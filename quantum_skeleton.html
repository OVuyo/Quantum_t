# Quantum_t – Quantum Finance Toolkit for MT5

Skeleton repo. Fill the placeholders and ship to MQL5 marketplace.

## Structure
- `core/`      → quantum finance primitives
- `api/`       → FastAPI endpoints
- `cli/`       → command-line interface
- `infra/`     → Docker / MT5 bridge / VPS scaling
- `tests/`     → pytest suite
- `examples/`  → Jupyter + Python demos
- `docs/`      → architecture & API docs

## Quick Start
```bash
pip install -r requirements.txt
docker compose up -d
python -m cli.main --help
